WallStreet Fighter

Wallstreet Fighter is another Phaser3 framework game! The making of this game taught me a lot about developing a playable game with a cool game loop in under a month. I had to learn so many new concepts to make the game interesting like AI, state machines, character design, sound design, animation sheets and all with experience in an engine I knew little about! I had a blast making this and I would love to make another beat-em-up in the future. Check out the game at itch.io below!
